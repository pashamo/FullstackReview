Forms

Form element always carries the action, method and id attributes. Methods of GET or POST. Use GET for short forms, just retreiving database information. Use POST for large forms, passwords, files and sensitive data. Get attaches the keys and values to the emd of the URL, POST sends the data in a HTTP header.
