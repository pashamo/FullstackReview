There are 6 levels of headings, from h1-h6

Items bolded with the "b" tag

Items italicized with "i" tag

Items superscripted with "sup" tag

Items subscripted with "sub" tag


Semantic Markup

  • "strong" used to indicate importance, displayed as bolded text
  • "em" used to emphasize text, displayed as italicized text
  • "blockquote"

    For longer quotes, can use the cite attribute

  • "q" For inline quotes, the browser adds quotation marks. Can also use the cite attribute
  • "abbr" ABBR for abbreviations
  • "cite" use this when referencing a piece of work, text is italicized
  • "dfn" for the defining instance of a new term, first use
  • "address" specific to author details
  • "ins/del" - inserted text in underline, deleted text with strikethrough
  • "s" - for content that is no longer valid, indicated with a strikethrough
